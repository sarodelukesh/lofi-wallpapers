# 🎧 Lofi Wallpapers — No Misses, Only Hits

This isn’t just another wallpaper dump.

Every single image in this repository has been **manually selected** — filtered, judged, and curated with one simple rule:

> **If it’s not good enough to use, it doesn’t exist here.**

---

## 🧠 Philosophy

The internet is flooded with “lofi” wallpapers that:

* look generic
* feel low-effort
* or just don’t hit the right vibe

This collection exists to eliminate that noise.

No bulk scraping.
No random dumps.
No “maybe this is fine” images.

Only wallpapers that pass a strict personal filter:

* Atmosphere ✔
* Aesthetic consistency ✔
* Actually usable daily ✔

---

## 🚫 What You Won’t Find Here

* Mid-tier wallpapers
* Repetitive variations
* Low-resolution junk
* “Technically lofi” but visually dead images

If it’s here, it’s because it *earned* its place.

---

## 🎯 What You Will Find

* Clean lofi environments
* Late-night / rainy mood setups
* Minimal but expressive compositions
* Wallpapers that actually **feel right** on your desktop

---

## 🛠 Usage

Clone or download:

```bash
git clone https://github.com/sarodelukesh/lofi-wallpapers.git
```

Then point your setup (Hyprland, etc.) to:

```text
~/Pictures/wallpapers/
```
